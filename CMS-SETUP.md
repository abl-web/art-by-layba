# CMS setup
1. Create a GitHub repo and upload this project.
2. In `public/admin/config.yml`, replace `abl-web/REPLACE-WITH-REPO-NAME` with your actual repo name.
3. Deploy the repo on Netlify.
4. Configure GitHub OAuth for Sveltia/Netlify, then visit `/admin`.

Note: The site components still need to be connected to the JSON files for live text/gallery updates. This package contains the CMS foundation and editable data files.
